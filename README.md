# Patavee Simple Portfolio

A professional trading portfolio and market insights platform built with React, TypeScript, and Vite. This project showcases trading performance, market analysis, and financial goals.

## 🚀 Features

- **TFEX Tracker:** Real-time monitoring and logging of TFEX trading performance.
- **Equity Curve:** Visual representation of trading growth and performance metrics.
- **Market Insights:** Daily summaries and analysis of Crypto, Stocks, and Commodities.
- **Life Goals Tracker:** A dedicated section for tracking long-term financial and personal objectives.
- **Calendar Integration:** Track trading days and significant market events.
- **Responsive Design:** Fully optimized for both desktop and mobile viewing.

## 🛠️ Tech Stack

- **Frontend:** React 19 (TypeScript)
- **Build Tool:** Vite
- **Styling:** Vanilla CSS
- **Routing:** React Router 7
- **Deployment:** GitHub Pages

## 📦 Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/pataveenu-max/Patavee-Simple-Portfolio.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
